Agent AI 프로젝트

LLM 모델과 CrewAI를 활용한 지능형 에이전트 기반 작업 자동화

프로젝트 개요

이 프로젝트는 다양한 LLM(OpenAI GPT, Claude, Gemini 등)과 CrewAI 프레임워크를 활용하여 에이전트(Agent), 태스크(Task), 크루(Crew)를 정의하고 실행하는 자동화 시스템을 구축합니다.

주요 작업은 Jupyter Lab 환경에서 진행되며, Miniconda 기반의 가상환경 설정과 .env 파일을 통한 API 키 관리가 포함됩니다.

작업 환경 및 설정

  • 작업 도구: Anaconda(Miniconda Prompt), Jupyter Lab
  • Python 버전: 3.10
  • 필수 라이브러리: crewai, langchain, dotenv 등
  • API 키: OpenAI (유료), GROQ (무료), Alpha Vantage
  • Alpha Vantage Key: 6Y4NE481I8KOT4N2

환경 구축 명령어

conda create -n crewai python=3.10
conda activate crewai
pip install crewai==0.28.8 crewai_tools==0.1.6 langchain_community==0.0.29 langchain_openai ipykernel langchain python-dotenv
python -m ipykernel install --user --name=crewai

.env 파일 주의사항: 실행 중인 파이썬 파일과 동일한 경로에 위치해야 합니다.

핵심 개념 요약

  • Agent: 역할, 목표, 배경, 도구 정의
  • Task: 에이전트가 수행할 실제 작업
  • Crew: 에이전트와 태스크의 집합, kickoff()로 실행
  • Memory: Short/Long/Entity 메모리 구조를 통한 문맥 유지
  • 기술: ReAct, Routing, Evaluator, Orchestrator-Worker 패턴 활용

참고 - 자동화된 블로그 콘텐츠 제작

콘텐츠 플래너, 콘텐츠 작가, 편집자 역할을 가진 세 에이전트를 정의하고, 주어진 주제에 대해 계획부터 작성, 편집까지 수행하도록 구성된 워크플로우.
