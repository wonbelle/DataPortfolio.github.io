🎨 Stable Diffusion LoRA 학습 프로젝트

조선시대 화풍을 반영한 AI 이미지 생성 실험

📌 프로젝트 개요

이 프로젝트는 Stable Diffusion v1.5 모델에 LoRA (Low-Rank Adaptation)를 적용하여, 조선 시대 화풍을 반영한 이미지를 생성하는 실험입니다.
참고한 화풍은 김홍도와 신사임당의 그림입니다.

🖼️ 학습 데이터

  • 원본 이미지 수: 약 20장 (김홍도, 신사임당 작품)
  • 데이터 증강 방식: Image Rotation (회전 증강)
  • 사용 기술: ImageDataGenerator(rotation_range=...) (Keras)

데이터가 부족했기 때문에 회전(Data Augmentation) 기법을 통해 학습 데이터를 증강했습니다. 다양한 각도의 이미지를 생성함으로써 모델이 화풍의 특징을 더 잘 일반화하도록 유도했습니다.

🛠 사용 도구 및 환경

  • 모델: Stable Diffusion v1.5
  • 학습 도구: LoRA (kohya-ss 기반)
  • 환경: Google Colab Pro
  • LoRA Rank: 4
  • 학습 시간: 약 1시간

📈 결과 및 회고

학습된 모델은 조선시대 특유의 붓 터치와 구도, 색감 등을 어느 정도 반영한 이미지를 생성할 수 있었습니다. 하지만 학습 데이터가 한정적이고 증강된 이미지의 다양성도 제한적이어서, 상품화 가능한 수준에는 도달하지 못했습니다.

향후 더 다양한 시도(다른 화가 포함, 더 다양한 Augmentation 기법 등)를 통해 퀄리티를 높일 계획입니다.
