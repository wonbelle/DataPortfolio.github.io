비트코인 자동매매 프로젝트

업비트 API와 기술적 지표를 활용한 자동매매 시스템 구현

프로젝트 개요

이 프로젝트는 업비트 API를 기반으로 실시간 가격 정보를 수신하고, 다양한 기술적 지표를 활용하여 자동으로 매수 및 매도 신호를 판단하는 시스템입니다.

매매 조건은 RSI, MACD, 이동평균선 등 다양한 지표를 조합하여 설정할 수 있으며, 테스트 시뮬레이션 및 실제 거래 테스트도 포함됩니다.

작업 환경 및 설정

  • 개발 언어: Python 3.x
  • 사용 API: 업비트 REST API
  • 라이브러리: pyupbit, pandas, numpy, talib, time 등
  • 기타: 백테스트를 위한 CSV 로그 저장 기능

주요 기능 및 로직

  • 실시간 가격 조회 및 캔들 데이터 수집
  • 기술적 지표 계산 (RSI, MACD, 이동평균 등)
  • 매수/매도 조건 판단 후 자동 주문
  • 잔고 확인, 수익률 계산, 로그 저장

테스트 시뮬레이션

전략별 백테스트 결과를 통해 각 전략의 수익률과 리스크를 분석하였습니다.

실제 매매는 소액 실거래 테스트로 운영되었으며, 안정성과 반응 속도를 고려한 로직을 지속 개선 중입니다.
